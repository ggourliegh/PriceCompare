# PriceCompare NZ

A grocery price comparison app for New Zealand's main supermarkets: **Pak'nSave**, **New World**, and **Woolworths**.

## Features

- **Weekly Specials** - Browse current deals across all three stores, filterable by store
- **Product Search** - Search products by name, brand, or category with full price comparison
- **Smart Shopping List** - Add items, see quantities, and get optimized store grouping with savings calculations
- **Price Scanner** - Take a photo of a product to find cheaper alternatives at other stores
- **Fridge to Recipe** - Snap your fridge contents (or select ingredients manually) and get recipe suggestions filtered by cooking time

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Zustand (state management with persistence)
- React Router v7
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

## Build
